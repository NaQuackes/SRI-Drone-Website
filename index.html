Project Overview

This engineering project develops a Smart Bridge Monitoring System using IoT sensors to monitor structural health in real time. The system detects vibrations, temperature changes, and strain, sending alerts when abnormalities are detected.

Project Specifications

Component Description
Microcontroller ESP32
Sensor Accelerometer, Temperature, Strain Gauge
Communication Wi-Fi
Power Solar + Battery Backup
